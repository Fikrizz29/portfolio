Nila-Pond

NilaPond is an informative website developed to support the cultivation and sales of tilapia owned by Tambak Nila Pak Nandang. This website provides complete information regarding high-quality tilapia cultivation practices, from the maintenance process to product distribution. One of the main features of NilaPond is the interactive reseller map, which makes it easier for prospective buyers to find tilapia reseller locations in the Bogor area. With this feature, users can quickly find the nearest distribution point and plan purchases more efficiently. This website was developed using front-end technologies such as HTML, CSS, and JavaScript, with a simple, responsive, and easy-to-use appearance by various groups.
